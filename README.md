#  Python Code Analyzer & AI Assistant (Kod Analizörü ve Yapay Zeka Asistanı)

Bu proje, geliştiricilerin kod kalitesini artırmak, endüstriyel standartlara uyumluluğu denetlemek ve yazılım süreçlerini optimize etmek amacıyla geliştirilmiş kapsamlı bir statik kod analizi aracıdır. Modern bir API mimarisi üzerinden çalışarak kod karmaşıklığını ölçer ve yapay zeka destekli iyileştirme önerileri sunar.

##  Kullanılan Teknolojiler ve Altyapı
* **Ana Framework:** FastAPI (Yüksek performanslı ve hızlı backend API yönetimi)
* **Statik Kod Analizi:** Flake8 (PEP 8 standartlarına uyumluluk ve hata denetimi)
* **Kod Karmaşıklığı Ölçümü:** Radon (Cyclomatic Complexity analizi ile kodun sürdürülebilirliğini ölçme)
* **Yapay Zeka Entegrasyonu:** Gemini API (Analiz edilen kodlar için otomatik optimizasyon ve temiz kod önerileri üreten modül)

##  Projenin Çözdüğü Temel Problemler
* **Kalite Kontrol Otomasyonu:** Yazılım ekiplerinin manuel olarak yaptığı kod gözden geçirme (code review) süreçlerini hızlandırır.
* **Sürdürülebilirlik (Maintainability):** Karmaşık ve ileride hata çıkarma riski yüksek kod bloklarını tespit eder.
* **Verimlilik:** Geliştiricilere anında geri bildirim vererek projelerin daha temiz bir mimariyle büyümesini sağlar.
